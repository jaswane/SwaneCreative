# **App Name**: Swane Landing

## Core Features:

- Core Message: Display key information about Swane Creative, highlighting its mission and values.
- Portfolio Showcase: Showcase a selection of Swane Creative's most compelling projects, using images, videos, and descriptions.
- Contact Information: Provide multiple contact options, including a contact form, email address, and social media links.
- AI Idea Generator: AI-powered "Idea Spark" tool which suggests creative project ideas based on user-defined themes or keywords.
- Client Testimonials: Feature testimonials from satisfied clients, reinforcing the value and reliability of Swane Creative.
- Call to Action: Include a clear call to action, encouraging visitors to request a quote or consultation.

## Style Guidelines:

- The color palette will be inspired by themes of creativity and forward-thinking, and feature a dark color scheme. The primary color is a deep sky blue (#64B5F6).
- The background color is a very dark, desaturated shade of blue to complement the primary (#121E29).
- The accent color is an analogous color to the primary, shifted slightly toward green, such as a vibrant light cyan (#76D2BB).
- Clean, modern typography to ensure readability and align with a professional image.
- Use minimalist, custom icons to represent services and key features.
- Clean and structured layout with clear visual hierarchy for intuitive navigation.
- Subtle animations to create an engaging and interactive user experience.
